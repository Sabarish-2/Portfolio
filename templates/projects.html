My Projects

Tax Calculator Website

I worked as an intern at The Money Concern, a startup, where I created a React JS tax calculator website. Taking into account factors like location, base pay, dearness allowance, HRA, LTA, and rent paid, this platform simplifies the process of calculating income. Even though my internship ended after a month, the financial company intends to improve the website for thorough tax computation.

Blue Ball Game

The Blue Ball Game, which I created using Unity and C#, is my first game development program project. It has two stages where players navigate a blue ball while gathering coins along the way, taking inspiration from old school favorites like the Red Ball game. My journey into game production began with this practical experience.

Dietricxs Application

This is an Android application written in Java using Android Studio. This is the application that I created as a project in college with a friend. This software lets you enter what you've eaten and then provides you with important statistics about your overall diet, such as how much protein, calories, vitamins, and so on you acquired. It offers a weekly history and login feature to save your progress. It saves all of your login and dietary information on Google Firebase.

Lunar Odyssey Game

The Lunar Odyssey Game is a team project that was created at a hackathon. It honors the accomplishment of India's Chandrayan 3 mission and was created in Unity using C#. Controlling the spacecraft, players must navigate lunar obstacles to get to water sources. I worked on the backend development of the game while my friend concentrated on level design, demonstrating how well we worked together.
